How To Play:


    Controls:

  • Shoot: Spacebar
  • Pass: Enter
  • Movement: Arrow Keys
  • Crossover: Left+Right
  • Stealing: If you touch the player with the ball you will steal it.

    • Rules:

  • There are four 3 minute quarters.
  • If the game is tied at the end then the game will continue until the next score.
